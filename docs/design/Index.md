# Index

evm-tx-engine 交易上链系统由三个模块组成:

1. 账号管理模块: 负责账户的创建, 私钥保存, 交易签名
2. 交易代付模块: 负责为交易或账户设置 gas 代付
3. 交易上链模块: 负责交易的组装, 上链, 监控, 重试, 状态更新

整个系统通过 REST-API 与外界交互. API 包含:

1. 账户创建接口, 账户查询接口
2. 交易上链任务创建接口, 交易状态接口
3. 其他: 系统监控, 交易调试

本系统支持 EVM 兼容链, 包括 以太坊和 Conflux eSpace 及其他兼容链.
